# 📊 SQL Total: De 0 a Avanzado en 12 Días

¡Bienvenido a mi repositorio de aprendizaje de SQL! Este espacio está dedicado a registrar mi progreso, proyectos diarios y código desarrollado durante el bootcamp intensivo **"SQL TOTAL - Domina Bases de Datos de 0 a Avanzado en 12 Días"** impartido por Federico Garay.

El objetivo de este repositorio es consolidar mis habilidades prácticas en diseño, manipulación, modelado y programación de bases de datos utilizando **MySQL**.

---

## 🛠️ Tecnologías y Herramientas
* **Motor de Base de Datos:** MySQL
* **Clientes / IDEs:** Visual Studio Code (con la extensión *Database Client*) y MySQL Workbench.
* **Lenguaje:** SQL (Structured Query Language).

---

## 📅 Bitácora del Programa (12 Días)

### 🔹 Fase 1: Fundamentos e Instalación (Días 1 - 2)
* **Día 1: Fundamentos**
  * Introducción a conceptos esenciales de bases de datos relacionales y sintaxis inicial de SQL.
* **Día 2: Instalación y Diseño Inicial 🚀 *(Proyecto Actual)***
  * Configuración del entorno de desarrollo local con MySQL.
  * **Proyecto Práctico:** Diseño y creación de mi primera base de datos relacional desde cero. 
  * *Caso de Estudio:* **Sistema de Cafetería** (`cafeteria_db`) estructurado con 3 tablas relacionadas (`clientes`, `productos`, `pedidos`) y poblado con datos de prueba utilizando llaves foráneas (`FOREIGN KEY`).

### 🔹 Fase 2: Consultas e Inserción Intensiva (Días 3 - 5)
* Escribir consultas constantes (básicas, intermedias y avanzadas).
* Uso avanzado de operadores, filtros (`WHERE`, `LIKE`, `IN`, `BETWEEN`) y funciones de agregación.

### 🔹 Fase 3: Diseño, Modelado y Programación (Días 6 - 8)
* Normalización de bases de datos.
* Creación de relaciones complejas, vistas (`VIEWS`), índices y restricciones de integridad.

### 🔹 Fase 4: Nivel Avanzado, Automatización y Nube (Días 9 - 12)
* Programación en SQL (Stored Procedures, Triggers, Funciones).
* Automatización de tareas de bases de datos.
* Integración de infraestructura de base de datos en la nube usando **AWS (Amazon Web Services)**.
* Preparación para entrevistas técnicas y resolución de desafíos del mundo real.
