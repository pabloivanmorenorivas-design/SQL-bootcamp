# SQL Total: Proyecto Día 1 - Fundamentos de Bases de Datos

¡Bienvenido al repositorio del **Día 1** de mi ruta de aprendizaje en SQL! Este proyecto marca el inicio de mi camino para dominar las bases de datos relacionales y el lenguaje SQL desde cero.

## 🎯 Objetivo del Día 1
El objetivo de esta primera sección fue comprender los pilares fundamentales de las bases de datos y entender cómo se almacena y estructura la información antes de empezar a escribir consultas complejas.

## 📚 Conceptos Aprendidos & Ejercitados
Durante este día, abordé y puse en práctica los siguientes temas clave:
* **¿Qué es una Base de Datos?**: Diferencia entre datos estructurados, semiestructurados y no estructurados.
* **Bases de Datos Relacionales (RDBMS)**: Cómo se organiza la información en tablas mediante filas (registros) y columnas (campos).
* **Introducción a SQL**: Qué es el lenguaje de consulta estructurado y por qué es el estándar de la industria para interactuar con datos.
* **Modelado Inicial**: Entendimiento conceptual de tablas, llaves primarias (Primary Keys) y la importancia de la integridad de los datos.
